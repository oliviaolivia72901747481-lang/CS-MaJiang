# 长沙麻将局域网联机体验 v0.8.2 手动验收 Checklist

本自检验收单用以指导测试人员在没有代码编辑器及开发背景的环境下，测试局域网多人对战功能。

---

## 💻 房主（电脑端）准备工作
- [ ] **1. 连接局域网或热点**
  - 房主电脑连接局域网 Wi-Fi，或手机开热点供电脑和其他手机连接，确保所有设备处于同一网段下。
- [ ] **2. 运行一键启动命令**
  - 在电脑终端运行以下命令一键启动前端和后端：
    ```bash
    npm run dev:online
    ```
- [ ] **3. 终端打印日志核验**
  - 检查控制台输出，确保出现以下格式的局域网访问提示（无报错）：
    ```text
    ==================================================
    🀄 长沙麻将多人联机服务端已成功启动！
    主机名称: XXXX
    Socket 服务地址 (本地): http://localhost:3001
    Socket 局域网服务地址:
      http://192.168.x.x:3001
    
    手机/局域网设备前端访问 URL:
      http://192.168.x.x:5173/?mode=online
    ==================================================
    ```

---

## 🏠 大厅与房间创建
- [ ] **4. 电脑端进入游戏**
  - 房主在浏览器打开 `http://localhost:5173`。
- [ ] **5. 输入昵称进入大厅**
  - 在昵称框输入“房主老张”，点击进入大厅，系统显示“已连接”。
- [ ] **6. 创建对局**
  - 点击“创建新对局”按钮。
- [ ] **7. 查看房间分享面板**
  - 房主界面成功渲染：
    - 右侧清晰的 **二维码分享组件**；
    - 二维码下方带有“复制链接”、“复制房间号”按钮；
    - 二维码上方显示“扫码加入对局”。

---

## 📱 手机端（玩家）加入流程
- [ ] **8. 手机扫码或输入链接**
  - 玩家手机（与电脑连接同一 Wi-Fi）使用微信或相机扫描电脑屏幕上的二维码，或者手动输入 `http://电脑IP:5173/?mode=online&roomId=XXXXXX`。
- [ ] **9. 自动带入房间号**
  - 手机端页面加载完成后，**房间号输入框中已自动填入 6 位数房间号**。
- [ ] **10. 输入昵称加入**
  - 手机玩家输入昵称（如“手机玩家李四”），点击“加入房间”，页面无延迟进入房间席位列表，且没有 `localhost` 错误。

---

## 🤖 AI 补位与游戏开始
- [ ] **11. 查看席位实时状态**
  - 房主及手机玩家座位列表上能够同时看到对方的在线状态及席位号。
- [ ] **12. AI 机器人补位**
  - 房主点击“🤖 机器人补位”按钮，房间空余位置瞬间被填入高级 AI，且 AI 头像显示为 `🤖 AI`。
- [ ] **13. 开始对局**
  - 房主点击“🎮 开始游戏”，全员（包括手机端）同时进入牌桌对局，没有任何死锁。

---

## 📶 手机端连接诊断自检
- [ ] **14. 手机访问 health 接口**
  - 手机浏览器直接打开 `http://电脑IP:3001/health`，页面能返回以下 JSON：
    ```json
    {"ok":true,"serverTime":...,"socketPort":3001,"activeRooms":1,"activeSockets":...}
    ```
- [ ] **15. 打开诊断面板**
  - 手机点击底部“🔍 打开局域网诊断自检面板”按钮，面板展开，显示：
    - 前端状态：`已加载`；
    - Socket 服务：`已连接`；
    - HTTP Health：`健康 (ok)`；
    - 当前访问 Host 与 Socket 真实连接地址。
- [ ] **16. 断网容灾测试**
  - 手机短暂断开 Wi-Fi，诊断面板能够实时感知并将 Socket 服务置为“未连接”，并显示故障排查建议。
  - 重新连上 Wi-Fi 后，状态自动变回“已连接”，且手牌状态无缝同步，无爆牌和作弊漏暗牌现象。
