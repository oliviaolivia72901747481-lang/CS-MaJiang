## 三、吃、碰、杠高亮及联动测试 (Highlights & Hover Interactions)

### 1. 手牌候选高亮
- [ ] 12. 轮到自己进行吃、碰、杠选择时，**确认手牌中参与该操作的候选组合具有明显的高亮（边框发光或背景色变化）**：
  - 吃牌组合：蓝色高亮类名 (`highlight-chi`)。
  - 碰牌组合：金色高亮类名 (`highlight-peng`)。
  - 杠牌组合：绿色高亮类名 (`highlight-gang`)。

### 2. 多选项悬停联动 (Hover Filtering)
- [ ] 13. 如果手牌中存在多种“吃”法（例如：别人打出 4万，自己有 2w+3w 与 5w+6w 两组吃法）：
  - 当鼠标**未悬停**任何多选按钮时，手牌中所有候选牌 (2w, 3w, 5w, 6w) **均显示高亮**。
  - 当鼠标**悬停**在“吃 2w 3w”的候选浮动选项上时，手牌中**仅高亮 2w 与 3w**，另两张 (5w, 6w) 变回普通样式。
  - 当鼠标**移开**选项后，手牌再次恢复全亮。
- [ ] 14. 如果手牌中存在多种“杠”法，悬停在特定杠法按钮时，手牌中**仅高亮对应的四张牌**。

### 3. 来源牌高亮
- [ ] 15. 别人打出的触发操作的那张牌（来源牌），在中心弃牌河中具有**红色发光与闪烁提示** (`highlight-source`)，使玩家能够一眼认清别人出的是哪张牌。
