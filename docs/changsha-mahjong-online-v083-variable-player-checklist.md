# 长沙麻将多人联网版 v0.8.3 手动验收 Checklist

本 Checklist 用于指导和记录对“v0.8.3：2–4 人可变人数局与可选 AI 模式”功能的局域网或本地手动验收测试。

## 验收流程说明

请在本地启动开发服务器，并通过多浏览器窗口/多移动设备模拟局域网多端接入进行验收。

- 服务端启动命令：`npm run dev:online` 或直接在网络大厅模块下运行
- 调试入口：`http://localhost:5173/` (或局域网 IP)

---

## 验收测试用例列表

### 一、2 人真人局直接开局
- [ ] 房主 A 创建房间（自动分配 Seat 0）
- [ ] 手机/窗口 B 通过扫码或输入 Room ID 加入房间（分配 Seat 1）
- [ ] 大厅界面正确显示：“当前玩家：2 真人，0 AI”
- [ ] 房主 A 点击【开始 2 人局】按钮，游戏成功开始
- [ ] **视觉校验**：游戏界面仅展示 2 个玩家大卡片（自己和对手），没有多余的空席位大卡片
- [ ] **发牌校验**：确认系统仅发牌给 Seat 0 和 Seat 1，Seat 2 和 3 的手牌不存在（牌数守恒为 108 - 14 - 13 = 81 张）
- [ ] **回合轮转**：打出牌后，回合在 Seat 0 和 Seat 1 之间轮流循环，无死锁
- [ ] **游戏完整度**：顺利完成至少 20 个出牌/吃碰杠动作或直至本局胡牌结算

### 二、3 人真人局直接开局
- [ ] 房主 A 创建房间（Seat 0）
- [ ] 玩家 B 加入房间（Seat 1）
- [ ] 玩家 C 加入房间（Seat 2）
- [ ] 大厅界面正确显示：“当前玩家：3 真人，0 AI”
- [ ] 房主 A 点击【开始 3 人局】按钮，游戏成功开始
- [ ] **视觉校验**：游戏界面仅展示自己与 2 个对手，空余席位完全隐藏，布局清晰
- [ ] **回合轮转**：出牌后回合在 Seat 0 -> 1 -> 2 -> 0 循环，跳过 Seat 3

### 三、真人 + AI 自由组合局
- [ ] 房主 A 创建房间（Seat 0），点击【添加 1 个 AI】，AI 成功进入 Seat 1
- [ ] 房主 A 点击【开始 2 人练习局】，游戏成功开始
- [ ] 确认游戏由 1 真人 + 1 AI 正常运行
- [ ] 重新创建房间，房主 A（Seat 0）和玩家 B（Seat 1）加入，房主点击【添加 1 个 AI】分配至 Seat 2
- [ ] 房主 A 点击【开始 3 人局】，确认 2 真人 + 1 AI 正常开局且结算

### 四、重连、断线与托管继承
- [ ] 2 人局中，玩家 B（Seat 1）刷新浏览器网页
- [ ] **重连校验**：玩家 B 网页刷新后通过 localStorage 缓存成功自动重连，恢复对手控制权，未被误踢
- [ ] 3 人局中，玩家 C（Seat 2）断开连接超过 30s
- [ ] **托管校验**：玩家 C 超过阈值后被 AI Trustee（托管）接管，游戏在托管状态下能继续轮转

### 五、结算与暗牌隔离安全
- [ ] 2 人局或 3 人局顺利胡牌，进入 Settlement 结算阶段
- [ ] **结算校验**：结算弹窗/界面只显示有分值变动的 activeSeat 玩家，inactiveSeats 既不得分也不失分，不参与中鸟
- [ ] **暗牌隔离校验**：在结算完成前，检查浏览器的 Network 传输数据，确认其他对手的暗手牌 instanceId 未被下发，没有信息泄露
