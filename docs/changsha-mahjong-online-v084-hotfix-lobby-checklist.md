# 长沙麻将多人联网版 v0.8.4-hotfix 联机大厅热修复手动验收 Checklist

本 Checklist 用于指引测试人员与开发人员对多人联机大厅的机器人逐个控制、扫码加入、默认首页进行手动回归。

## 一、服务启动与默认首页
- [ ] 房主执行 `npm run dev:online` 并确保控制台打印后端 Socket 服务器（端口 3001）和前端服务（端口 5173）。
- [ ] 房主在电脑浏览器中直接打开 `http://localhost:5173/`。
- [ ] **确认默认直接进入“多人联机大厅”，而非之前的单机陪练模式。**
- [ ] 确认页面顶部出现模式切换栏，且点击“单机陪练模式”可以成功切换。
- [ ] 确认从单机陪练模式点击“多人联机模式”可以成功切回。

## 二、大厅机器人逐个添加与移除
- [ ] 房主在大厅中点击“创建新对局”，进入房间等待界面。
- [ ] 确认此时界面显示 `当前状态: 1 真人 + 0 AI (共 1 人)`，且底部的开始游戏按钮显示 `🎮 开始游戏 (请添加机器人或等待玩家)` 并处于禁用状态。
- [ ] 点击控制区中的 `[➕ 添加 1 个机器人]` 按钮。
- [ ] **确认只添加了 1 个 AI (位置 1)，而非一次性补满。** 当前状态显示 `1 真人 + 1 AI (共 2 人)`。
- [ ] 确认座位 1 右侧显示有红色 `[移除机器人]` 按钮。
- [ ] 再点击一次 `[➕ 添加 1 个机器人]` 确认在座位 2 上又增加了一个 AI。
- [ ] 点击座位 1 右侧的 `[移除机器人]`。
- [ ] **确认只有该席位的机器人被成功踢出房间，且席位重新变为“空席”和“添加机器人”。**
- [ ] 房主座位（座位 0，真人）右侧不显示“移除机器人”按钮。
- [ ] 点击控制区中的 `[🤖 补到 4 人]` 按钮。
- [ ] 确认大厅席位一次性补满（包含 1 个真人 + 3 个 AI）。
- [ ] 房主点击 `[🎮 开始 4 人局]`，确认对局可以顺利开始，且开始后任何 AI 添加与移除按钮全部消失/被禁用。

## 三、真实手机局域网扫码加入
- [ ] 房主在 localhost 页面创建房间（如房间号 `482910`）。
- [ ] 确认大厅右侧展示二维码及“扫码加入对局”指示。
- [ ] **确认二维码下方的“分享链接”不是 localhost / 127.0.0.1，而是真实的局域网 IP（例如 `http://192.168.1.100:5173/?mode=online&roomId=482910`）。**
- [ ] 测试手机（需连接在同一个 Wi-Fi 局域网下）扫描该二维码。
- [ ] 手机浏览器打开链接，**确认自动进入多人联机大厅，并自动 prefill 填入房间号 `482910`。**
- [ ] **如果手机第一次访问，确认不会自动发送加入请求，页面提示“加入房间 482910”及“请输入昵称后加入房间”说明。**
- [ ] 手机端输入昵称（例如 `Bob`）并点击“进入大厅”，确认成功发送加入请求并作为座位 1 加入房间。
- [ ] 房主的电脑端和手机端同步刷新席位状态。
- [ ] 控制台无任何报错信息。
